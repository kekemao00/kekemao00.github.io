常用参数:

  • --rm 说明容器用完就销毁,即停止后就删除了,就没了

  • -d 容器在后台运行

  • -e 给新创建的数据库设置 root 用户的密码

  • --name 将容器命名为 mysql_3306 ,建议命名后面根端口号

  • -v 把容器的 /var/lib/mysql 映射到创建的数据卷上

  • -p 前面宿主端口,后面组件端口, 把所有对宿主机 3306 端口的 tcp 报文转发到容器的 3306 端口
